### 3.常变量
 - 定义常量使用val
    + val a: Int = 1  
    + val b = 1       // 系统自动推断变量类型为Int
    + val c: Int      // 如果不在声明时初始化则必须提供变量类型
    + c = 1           // 明确赋值
    + var x = 5        // 系统自动推断变量类型为Int
    + x += 1           // 变量可修改
