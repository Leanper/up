###  1.1 介绍安装

- virtualenv通过创建一个虚拟化的python运行环境，将我们所需的依赖安装进去的，不同项目之间相互不干扰
- pip install virtualenv 
- 若安装失败通过 pip3 install virtualenv -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
